\doxysubsection{Detailed Description}
\mbox{\hyperlink{classgko_1_1log_1_1Loggable}{Loggable}} class is an interface which should be implemented by classes wanting to support logging. 

For most cases, one can rely on the \mbox{\hyperlink{classgko_1_1log_1_1EnableLogging}{Enable\+Logging}} mixin which provides a default implementation of this interface.
