\doxysubsection{Detailed Description}
This class stores additional context for creating Ginkgo objects from configuration files. 

The context can contain user-\/provided objects that derive from the following base types\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{classgko_1_1LinOp}{Lin\+Op}}
\item \mbox{\hyperlink{classgko_1_1LinOpFactory}{Lin\+Op\+Factory}}
\item Criterion\+Factory
\end{DoxyItemize}

Additionally, users can provide mappings from a configuration (provided as a pnode) to user-\/defined types that are derived from \mbox{\hyperlink{classgko_1_1LinOpFactory}{Lin\+Op\+Factory}}
