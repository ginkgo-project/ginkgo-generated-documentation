\subsubsection*{template$<$typename Value\+Type = default\+\_\+precision$>$\newline
class gko\+::stop\+::\+Relative\+Residual\+Norm$<$ Value\+Type $>$}

The \mbox{\hyperlink{classgko_1_1stop_1_1RelativeResidualNorm}{Relative\+Residual\+Norm}} class is a stopping criterion which stops the iteration process when the residual norm is below a certain threshold relative to the norm of the right-\/hand side, i.\+e. 

when norm(residual) / norm(right\+\_\+hand\+\_\+side) $<$ threshold. For better performance, the checks are run thanks to kernels on the executor where the algorithm is executed.

\begin{DoxyNote}{Note}
To use this stopping criterion there are some dependencies. The constructor depends on {\ttfamily b} in order to compute the norm of the right-\/hand side. If this is not correctly provided, an exception \+::gko\+::\+Not\+Supported() is thrown.
\end{DoxyNote}
