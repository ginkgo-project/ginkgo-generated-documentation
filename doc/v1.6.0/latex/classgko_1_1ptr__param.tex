\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class gko\+::ptr\+\_\+param$<$ T $>$}

This class is used for function parameters in the place of raw pointers. 

Pointer parameters should be used for everything that does not involve transfer of ownership. It can be converted to from raw pointers, shared pointers and unique pointers of the specified type or any derived type. This allows functions to be called without having to use \mbox{\hyperlink{namespacegko_a014ec51e675c425bb83908361f630450}{gko\+::lend}} or calling .\mbox{\hyperlink{classgko_1_1ptr__param_a815e682f273679b99cd632a83c974057}{get()}} for every pointer argument. It probably has no use outside of function parameters, as it is immutable.
