\subsubsection*{template$<$typename Value\+Type = default\+\_\+precision$>$\newline
class gko\+::matrix\+::\+Diagonal$<$ Value\+Type $>$}

This class is a utility which efficiently implements the diagonal matrix (a linear operator which scales a vector row wise). 

Objects of the \mbox{\hyperlink{classgko_1_1matrix_1_1Diagonal}{Diagonal}} class always represent a square matrix, and require one array to store their values.
