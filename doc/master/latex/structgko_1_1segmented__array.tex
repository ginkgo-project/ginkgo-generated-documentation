\subsubsection*{template$<$typename T$>$\newline
struct gko\+::segmented\+\_\+array$<$ T $>$}

A minimal interface for a segmented array. 

The segmented array is stored as a flat buffer with an offsets array. The segment {\ttfamily i} contains the index range {\ttfamily \mbox{[}offset\mbox{[}i\mbox{]}, offset\mbox{[}i + 1\mbox{]})} of the flat buffer.
