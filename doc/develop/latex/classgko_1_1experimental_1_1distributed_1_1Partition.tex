\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Local\+Index\+Type = int32, typename Global\+Index\+Type = int64$>$\newline
class gko\+::experimental\+::distributed\+::\+Partition$<$ Local\+Index\+Type, Global\+Index\+Type $>$}

Represents a partition of a range of indices \mbox{[}0, size) into a disjoint set of parts. 

The partition is stored as a set of consecutive ranges \mbox{[}begin, end) with an associated part ID and local index (number of indices in this part before {\ttfamily begin}). Global indices are stored as 64 bit signed integers (int64), part-\/local indices use Local\+Index\+Type, Part I\+Ds use 32 bit signed integers (int).

For example, consider the interval \mbox{[}0, 13) that is partitioned into the following ranges\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{[0,3), [3, 6), [6, 8), [8, 10), [10, 13).}
\end{DoxyCode}
 These ranges are distributed on three part with\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{p\_0 = [0, 3) + [6, 8) + [10, 13),}
\DoxyCodeLine{p\_1 = [3, 6),}
\DoxyCodeLine{p\_2 = [8, 10).}
\end{DoxyCode}
 The part ids can be queried from the \mbox{\hyperlink{classgko_1_1experimental_1_1distributed_1_1Partition_a0533d86b3cf4830b5250366b18bfa842}{get\+\_\+part\+\_\+ids}} array, and the ranges are represented as offsets, accessed by \mbox{\hyperlink{classgko_1_1experimental_1_1distributed_1_1Partition_a922e3a6b00fa1d9d6c9988c8f3de31e8}{get\+\_\+range\+\_\+bounds}}, leading to the offset array\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{r = [0, 3, 6, 8, 10, 13]}
\end{DoxyCode}
 so that individual ranges are given by {\ttfamily \mbox{[}r\mbox{[}i\mbox{]}, r\mbox{[}i + 1\mbox{]})}. Since each part may be associated with multiple ranges, it is possible to get the starting index for each range that is local to the owning part, see \mbox{\hyperlink{classgko_1_1experimental_1_1distributed_1_1Partition_a28993e20e1a600f0d130ccaf2187aaa8}{get\+\_\+range\+\_\+starting\+\_\+indices}}. These indices can be used to easily iterate over part local data. For example, the above partition has the following starting indices 
\begin{DoxyCode}{0}
\DoxyCodeLine{starting\_index[0] = 0,}
\DoxyCodeLine{starting\_index[1] = 0,}
\DoxyCodeLine{starting\_index[2] = 3,  \textcolor{comment}{// second range of part 1}}
\DoxyCodeLine{starting\_index[3] = 0,}
\DoxyCodeLine{starting\_index[4] = 5,  \textcolor{comment}{// third range of part 1}}
\end{DoxyCode}
 which you can use to iterate only over the the second range of part 1 (the third global range) with 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordflow}{for}(\textcolor{keywordtype}{int} i = 0; i < r[3] -\/ r[2]; ++i)\{}
\DoxyCodeLine{  data[starting\_index[2] + i] = val;}
\DoxyCodeLine{\}}
\end{DoxyCode}

\hline
