class gko\+::matrix\+::\+Fbcsr$<$ Value\+Type, Index\+Type $>$}

Fixed-\/block compressed sparse row storage matrix format. 

F\+B\+C\+SR is a matrix format meant for matrices having a natural block structure made up of small, dense, disjoint blocks. It is similar to C\+SR \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{classgko_1_1matrix_1_1Csr}{Csr}}. However, unlike \mbox{\hyperlink{classgko_1_1matrix_1_1Csr}{Csr}}, each non-\/zero location stores a small dense block of entries having a constant size. This reduces the number of integers that need to be stored in order to refer to a given non-\/zero entry, and enables efficient implementation of certain block methods.
\end{DoxySeeAlso}
The block size is expected to be known in advance and passed to the constructor.

\begin{DoxyNote}{Note}
The total number of rows and the number of columns are expected to be divisible by the block size.
\end{DoxyNote}
The nonzero elements are stored in a 1D array row-\/wise, and accompanied with a row pointer array which stores the starting index of each block-\/row. An additional block-\/column index array is used to identify the block-\/column of each nonzero block.

The \mbox{\hyperlink{classgko_1_1matrix_1_1Fbcsr}{Fbcsr}} \mbox{\hyperlink{classgko_1_1LinOp}{Lin\+Op}} supports different operations\+:
