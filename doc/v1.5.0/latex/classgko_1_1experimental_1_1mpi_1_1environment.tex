\doxysubsection{Detailed Description}
Class that sets up and finalizes the M\+PI environment. 

This class is a simple R\+A\+II wrapper to M\+P\+I\+\_\+\+Init and M\+P\+I\+\_\+\+Finalize.

M\+P\+I\+\_\+\+Init must have been called before calling any M\+PI functions.

\begin{DoxyNote}{Note}
If M\+P\+I\+\_\+\+Init has already been called, then this class should not be used. 
\end{DoxyNote}
