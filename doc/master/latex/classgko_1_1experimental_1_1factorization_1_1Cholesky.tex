class gko\+::experimental\+::factorization\+::\+Cholesky$<$ Value\+Type, Index\+Type $>$}

Computes a \mbox{\hyperlink{classgko_1_1experimental_1_1factorization_1_1Cholesky}{Cholesky}} factorization of a symmetric, positive-\/definite sparse matrix. 

This \mbox{\hyperlink{classgko_1_1LinOpFactory}{Lin\+Op\+Factory}} returns a \mbox{\hyperlink{classgko_1_1experimental_1_1factorization_1_1Factorization}{Factorization}} storing the L and L$^\wedge$H factors for the provided system matrix in \mbox{\hyperlink{classgko_1_1matrix_1_1Csr}{matrix\+::\+Csr}} format. If no symbolic factorization is provided, it will be computed first.
