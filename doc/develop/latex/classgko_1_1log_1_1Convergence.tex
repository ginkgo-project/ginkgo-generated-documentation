\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Value\+Type = default\+\_\+precision$>$\newline
class gko\+::log\+::\+Convergence$<$ Value\+Type $>$}

\mbox{\hyperlink{classgko_1_1log_1_1Convergence}{Convergence}} is a \mbox{\hyperlink{classgko_1_1log_1_1Logger}{Logger}} which logs data strictly from the {\ttfamily criterion\+\_\+check\+\_\+completed} event. 

The purpose of this logger is to give a simple access to standard data generated by the solver once it has stopped with minimal overhead.

This logger also computes the residual norm from the residual when the residual norm was not available. This can add some slight overhead.
