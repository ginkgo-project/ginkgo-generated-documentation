Detailed Description

template<typename ValueType = default_precision, typename IndexType = int32>
class gko::matrix::BatchEll< ValueType, IndexType >

BatchEll is a matrix format which stores only the nonzero coefficients by compressing each row of the matrix (compressed sparse row format).

Each of the individual batches are stored in a CSR matrix.

Template Parameters
ValueTypeprecision of matrix elements
IndexTypeprecision of matrix indexes
Note
Currently, BatchEll can store matrices with batch entries that have the same sparsity pattern, but different values.
