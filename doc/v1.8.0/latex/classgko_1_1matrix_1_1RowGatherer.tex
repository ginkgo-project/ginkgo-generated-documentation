\mbox{\hyperlink{classgko_1_1matrix_1_1RowGatherer}{Row\+Gatherer}} is a matrix \char`\"{}format\char`\"{} which stores the gather indices arrays which can be used to gather rows to another matrix. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Index\+Type} & precision of rowgatherer array indices.\\
\hline
\end{DoxyTemplParams}
\begin{DoxyNote}{Note}
This format is used mainly to allow for an abstraction of the rowgatherer and provides the user with an apply method which calls the respective \mbox{\hyperlink{classgko_1_1matrix_1_1Dense}{Dense}} rowgatherer operation. As such it only stores an array of the rowgatherer indices. 
\end{DoxyNote}
