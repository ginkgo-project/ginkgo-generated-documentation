\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Value\+Type = default\+\_\+precision$>$\newline
class gko\+::batch\+::solver\+::\+Cg$<$ Value\+Type $>$}

\mbox{\hyperlink{classgko_1_1batch_1_1solver_1_1Cg}{Cg}} or the Conjugate Gradient is a Krylov subspace solver. 

It is a short recurrence solver that is generally used to solve linear systems with S\+PD matrices.

This solver solves a batch of linear systems using the \mbox{\hyperlink{classgko_1_1batch_1_1solver_1_1Cg}{Cg}} algorithm. Each linear system in the batch can converge independently.

Unless otherwise specified via the {\ttfamily preconditioner} factory parameter, this implementation does not use any preconditioner by default. The type of tolerance (absolute or relative) and the maximum number of iterations to be used in the stopping criterion can be set via the factory parameters.

\begin{DoxyNote}{Note}
The tolerance check is against the internal residual computed within the solver process. This implicit (internal) residual can diverge from the true residual ($\vert$$\vert$b -\/ Ax$\vert$$\vert$). A posterori checks (by computing the true residual, $\vert$$\vert$b -\/ Ax$\vert$$\vert$) are recommended to ensure that the solution has converged to the desired tolerance.
\end{DoxyNote}
