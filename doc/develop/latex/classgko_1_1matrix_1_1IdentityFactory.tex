\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Value\+Type = default\+\_\+precision$>$\newline
class gko\+::matrix\+::\+Identity\+Factory$<$ Value\+Type $>$}

This factory is a utility which can be used to generate \mbox{\hyperlink{classgko_1_1matrix_1_1Identity}{Identity}} operators. 

The factory will generate the \mbox{\hyperlink{classgko_1_1matrix_1_1Identity}{Identity}} matrix with the same dimension as the passed in operator. It will throw an exception if the operator is not square.
