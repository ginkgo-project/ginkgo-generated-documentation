class gko\+::experimental\+::factorization\+::\+Lu$<$ Value\+Type, Index\+Type $>$}

Computes an LU factorization of a sparse matrix. 

This \mbox{\hyperlink{classgko_1_1LinOpFactory}{Lin\+Op\+Factory}} returns a \mbox{\hyperlink{classgko_1_1experimental_1_1factorization_1_1Factorization}{Factorization}} storing the L and U factors for the provided system matrix in \mbox{\hyperlink{classgko_1_1matrix_1_1Csr}{matrix\+::\+Csr}} format. If no symbolic factorization is provided, it will be computed first.
