class gko\+::multigrid\+::\+Pgm$<$ Value\+Type, Index\+Type $>$}

Parallel graph match (\mbox{\hyperlink{classgko_1_1multigrid_1_1Pgm}{Pgm}}) is the aggregate method introduced in the paper M. 

Naumov et al., \char`\"{}\+Amg\+X\+: A Library for G\+P\+U Accelerated Algebraic
\+Multigrid and Preconditioned Iterative Methods\char`\"{}. Current implementation only contains size = 2 version.

\mbox{\hyperlink{classgko_1_1multigrid_1_1Pgm}{Pgm}} creates the aggregate group according to the matrix value not the structure. \mbox{\hyperlink{classgko_1_1multigrid_1_1Pgm}{Pgm}} gives two steps (one-\/phase handshaking) to group the elements. 1\+: get the strongest neighbor of each unaggregated element. 2\+: group the elements whose strongest neighbor is each other. repeating until reaching the given conditions. After that, the un-\/aggregated elements are assigned to an aggregated group or are left alone.
