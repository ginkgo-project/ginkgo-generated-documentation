\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Value\+Type = default\+\_\+precision$>$\newline
class gko\+::log\+::\+Papi$<$ Value\+Type $>$}

\mbox{\hyperlink{classgko_1_1log_1_1Papi}{Papi}} is a \mbox{\hyperlink{classgko_1_1log_1_1Logger}{Logger}} which logs every event to the P\+A\+PI software. 

Thanks to this logger, applications which interface with P\+A\+PI can access Ginkgo internal data through P\+A\+PI. For an example of usage, see examples/papi\+\_\+logging/papi\+\_\+logging.\+cpp

The logged values for each event are the following\+:
\begin{DoxyItemize}
\item all allocation events\+: number of bytes per executor
\item all free events\+: number of calls per executor
\item copy\+\_\+started\+: number of bytes per executor from (to), in copy\+\_\+started\+\_\+from (respectively copy\+\_\+started\+\_\+to).
\item copy\+\_\+completed\+: number of bytes per executor from (to), in copy\+\_\+completed\+\_\+from (respectively copy\+\_\+completed\+\_\+to).
\item all polymorphic objects and operation events\+: number of calls per executor
\item all apply events\+: number of calls per \mbox{\hyperlink{classgko_1_1LinOp}{Lin\+Op}} (argument \char`\"{}\+A\char`\"{}).
\item all factory events\+: number of calls per factory
\item criterion\+\_\+check\+\_\+completed event\+: the residual norm is stored in a record (per criterion)
\item iteration\+\_\+complete event\+: the number of iteration is counted (per solver)
\end{DoxyItemize}
