\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Value\+Type$>$\newline
class gko\+::experimental\+::mpi\+::window$<$ Value\+Type $>$}

This class wraps the M\+P\+I\+\_\+\+Window class with R\+A\+II functionality. 

Different create and lock type methods are setup with enums.

M\+P\+I\+\_\+\+Window is primarily used for one sided communication and this class provides functionalities to fence, lock, unlock and flush the communication buffers.
