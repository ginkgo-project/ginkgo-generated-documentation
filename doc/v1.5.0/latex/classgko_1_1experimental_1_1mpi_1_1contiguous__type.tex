\doxysubsection{Detailed Description}
A move-\/only wrapper for a contiguous M\+P\+I\+\_\+\+Datatype. 

The underlying M\+P\+I\+\_\+\+Datatype is automatically created and committed when an object of this type is constructed, and freed when it is destructed.
