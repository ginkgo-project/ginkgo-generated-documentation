class gko\+::experimental\+::solver\+::\+Direct$<$ Value\+Type, Index\+Type $>$}

A direct solver based on a factorization into lower and upper triangular factors (with an optional diagonal scaling). 

The solver is built from the Factorization returned by the provided \mbox{\hyperlink{classgko_1_1LinOpFactory}{Lin\+Op\+Factory}}.
