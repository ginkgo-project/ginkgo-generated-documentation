class gko\+::matrix\+::\+Permutation$<$ Index\+Type $>$}

\mbox{\hyperlink{classgko_1_1matrix_1_1Permutation}{Permutation}} is a matrix \char`\"{}format\char`\"{} which stores the row and column permutation arrays which can be used for re-\/ordering the rows and columns a matrix. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Index\+Type} & precision of permutation array indices.\\
\hline
\end{DoxyTemplParams}
\begin{DoxyNote}{Note}
This format is used mainly to allow for an abstraction of the permutation/re-\/ordering and provides the user with an apply method which calls the respective \mbox{\hyperlink{classgko_1_1LinOp}{Lin\+Op}}\textquotesingle{}s permute operation if the respective \mbox{\hyperlink{classgko_1_1LinOp}{Lin\+Op}} implements the \mbox{\hyperlink{classgko_1_1Permutable}{Permutable}} interface. As such it only stores an array of the permutation indices. 
\end{DoxyNote}
