\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Local\+Index\+Type, typename Global\+Index\+Type = int64$>$\newline
struct gko\+::experimental\+::distributed\+::index\+\_\+map$<$ Local\+Index\+Type, Global\+Index\+Type $>$}

This class defines mappings between global and local indices. 

Given an index space $I = [0, \dots, N)$ that is partitioned into $P$ disjoint subsets $I_k, k = 1, \dots, P$, this class defines for each subset an extended global index set $\hat{I}_k \supset I_K$. The extended index set contains the global indices owned by part $k$, as well as remote indices $R_k = \hat{I}_k \setminus I_k$, which are also accessed by part $k$, but owned by parts $l \neq k$. At the core, this class provides mappings from the global index space $I$ into different local index spaces. The combined local index space (\mbox{\hyperlink{namespacegko_1_1experimental_1_1distributed_a3d50932cfebdde844b88d2c82b96947ba507d922378ecab8e10b81dcbc2ff8bf3}{index\+\_\+space\+::combined}}) is then defined as $[0, \dots, |\hat{I}_k|)$. Additionally, the combined index space can be separated into locally owned (\mbox{\hyperlink{namespacegko_1_1experimental_1_1distributed_a3d50932cfebdde844b88d2c82b96947baf5ddaf0ca7929578b408c909429f68f2}{index\+\_\+space\+::local}}) and non-\/locally owned (\mbox{\hyperlink{namespacegko_1_1experimental_1_1distributed_a3d50932cfebdde844b88d2c82b96947ba9182221a9ec0a65321d5a206c9c174ca}{index\+\_\+space\+::non\+\_\+local}}). The locally owned indices are defined as $[0, \dots, |I_k|)$, and the non-\/locally owned as $[0, \dots, |R_k|)$. With these index sets, the following mappings are defined\+:


\begin{DoxyItemize}
\item $c_k : \hat{I}_k \mapsto [0, \dots, |\hat{I}_k|)$ which maps global indices into the combined/full local index space (denoted as \mbox{\hyperlink{namespacegko_1_1experimental_1_1distributed_a3d50932cfebdde844b88d2c82b96947ba507d922378ecab8e10b81dcbc2ff8bf3}{index\+\_\+space\+::combined}}),
\item $l_k: I_k \mapsto [0, \dots, |I_k|)$ which maps global indices into the locally owned index space (denoted as \mbox{\hyperlink{namespacegko_1_1experimental_1_1distributed_a3d50932cfebdde844b88d2c82b96947baf5ddaf0ca7929578b408c909429f68f2}{index\+\_\+space\+::local}}),
\item $r_k: R_k \mapsto [0, \dots, |R_k|)$ which maps global indices into the non-\/locally owned index space (denoted as \mbox{\hyperlink{namespacegko_1_1experimental_1_1distributed_a3d50932cfebdde844b88d2c82b96947ba9182221a9ec0a65321d5a206c9c174ca}{index\+\_\+space\+::non\+\_\+local}}).
\end{DoxyItemize}

The required map can be selected by passing the appropriate type of an index\+\_\+space.

The index map for $I_k$ has no knowledge about any other index maps for $I_l, l \neq k$. In particular, any global index passed to the {\ttfamily map\+\_\+to\+\_\+local} map that is not part of the specified index space, will be mapped to an invalid\+\_\+index.

The set $R_k = \hat{I}_k \setminus I_k$ can also be written as the union of the intersection of $\hat{I}_k$ with other disjoint sets $I_l, l \neq k$, i.\+e. $R_k = \bigcup_{j \neq k} \hat{I}_k \cap I_j = \bigcup_{j \neq k} R_{k,j}$. The set $R_{k,j}$ can then be mapped by $l_j$ to get the local indices wrt. part $j$.
