Convert the given \mbox{\hyperlink{classgko_1_1LinOp}{Lin\+Op}} from experimental\+::distributed\+::\+Vector$<$...$>$ to experimental\+::distributed\+::\+Vector$<$\+Value\+Type$>$. 

The conversion tries to convert the input \mbox{\hyperlink{classgko_1_1LinOp}{Lin\+Op}} to all Dense types with value type recursively reachable by next\+\_\+precision$<$...$>$ starting from the Value\+Type template parameter. This means that all real-\/to-\/real and complex-\/to-\/complex conversions for default precisions are being considered. If the input matrix is non-\/const, the contents of the modified converted object will be converted back to the input matrix when the returned object is destroyed. This may lead to a loss of precision!
